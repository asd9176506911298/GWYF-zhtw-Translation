# Release v1.0.4

完整模組包內容：

- BepInEx runtime
- XUnity AutoTranslator
- XUnity ResourceRedirector
- 目前使用中的 zh-TW 翻譯文本與貼圖
- GWYF 自製插件：
  - GWYFUnifiedPanel
  - SteamIdJoiner
  - GwyfModeration
  - GWYFTranslationUpdater
  - GWYFHostedPanel
- 預設已指向：
  - https://github.com/XoF-eLtTiL/GWYF-zhtw-Translation

壓縮檔：

- `releases/GWYF-zhtw-Translation-Pack-v1.0.4.zip`

安裝方式：

1. 解壓到遊戲根目錄
2. 覆蓋既有檔案
3. 啟動遊戲
4. `F10` 開啟整合面板

翻譯更新：

- updater 會使用 repo 的 raw manifest 與 translations 路徑
- 後續只要更新翻譯、重建 `manifest.txt`、push 即可
