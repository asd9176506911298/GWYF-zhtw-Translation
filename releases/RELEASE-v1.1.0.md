# Release v1.1.0

公開內容：

- BepInEx runtime
- XUnity AutoTranslator
- XUnity ResourceRedirector
- `Gamble With Your Friends` zh-TW 翻譯文本與貼圖
- GWYFTranslationUpdater 自動更新翻譯模組

不包含：

- 私人的加入方法
- 私人的伺服器列表 / Lobby 工具
- 私人的 moderation / 管理功能
- 其他私人插件

安裝方式：

1. 解壓到遊戲根目錄
2. 覆蓋既有檔案
3. 啟動遊戲
4. updater 會在啟動後自動檢查翻譯更新
